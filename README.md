# Smart-Door
Judul Proyek:
Implementasi Sistem Pakar Fordward Chainning untuk Pengendalian Akses Pintu Cerdas dengan Notifikasi Telegram menggunakan ESP32

Deskripsi Singkat:
Sistem keamanan akses pintu konvensional memiliki keterbatasan dalam pengambilan keputusan 
secara adaptif terhadap berbagai skenario ancaman. Penelitian ini merancang dan 
mengimplementasikan sistem pakar berbasis metode Forward Chaining untuk mengendalikan 
akses pintu cerdas yang terintegrasi dengan perangkat ESP32 dan notifikasi Telegram. Basis 
pengetahuan sistem dibangun dari aturan IF-THEN yang diperoleh melalui proses akuisisi 
pengetahuan dari pakar keamanan.
